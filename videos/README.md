# Video Files

Place your video files here (.mp4, .webm)

## File Naming

The demo expects:
- `movie-clip-sample.mp4` - A famous movie scene for "Name That Scene"

## Tips

1. Keep video clips short (10-60 seconds)
2. Use MP4 format for best compatibility
3. Compress videos to keep file sizes manageable
4. Use descriptive filenames

## Example Structure

```
videos/
├── movie-clip-sample.mp4
├── titanic-ending.mp4
├── matrix-bullet-time.mp4
└── forrest-gump-bench.mp4
```
