# Audio Files

Place your audio files here (.mp3, .wav, .ogg)

## File Naming

The demo expects:
- `movie-score-sample.mp3` - A movie soundtrack clip for "Guess the Movie"

## Tips

1. Keep audio clips short (5-30 seconds)
2. Use descriptive filenames
3. Supported formats: MP3, WAV, OGG, WebM

## Example Structure

```
audio/
├── movie-score-sample.mp3
├── star-wars-theme.mp3
├── jurassic-park-theme.mp3
└── jaws-theme.mp3
```
